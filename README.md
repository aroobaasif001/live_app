# Havatai

**Havatai** is a modern real-time live streaming and interactive social entertainment application built with Flutter, designed to deliver an engaging and community-driven user experience.

The platform combines **live streaming, audience interaction, virtual gifting, real-time communication, social engagement, and creator-focused features** into one scalable mobile ecosystem.

## Key Features

### Live Streaming
- Users can start live broadcasts in real time
- Viewers can join live rooms and interact instantly
- Smooth and immersive live streaming experience
- Designed for creator and audience engagement

### Audience Interaction
- Real-time comments and reactions during live sessions
- Interactive communication between streamers and viewers
- Community-focused engagement experience

### Virtual Gifting System
- Users can send virtual gifts to streamers during live sessions
- Supports creator engagement and monetization flows
- Enhances audience participation inside live rooms

### Real-Time Chat
- Private user-to-user messaging
- Fast and responsive communication system
- Social interaction beyond live streaming sessions

### Audio & Video Calling
- One-to-one audio calling
- One-to-one video calling
- Seamless in-app communication between users

### Reels & Short Video Content
- Reels-style short video browsing experience
- Entertainment-focused content discovery
- Designed to improve user retention and engagement

### Social Features
- User profiles and social discovery
- Connect and interact with other users
- Community-driven entertainment ecosystem

## Technology Stack & Integrations

- **Flutter & Dart** — Cross-platform mobile application development
- **Agora / Real-Time Streaming SDK** — Live streaming and communication support
- **Firebase** — Authentication, database, storage, and backend services
- **Real-Time Chat System** — Instant messaging and live room interaction
- **Push Notifications** — User engagement and activity updates
- **Virtual Coins & Gifting System** — Live gifting and engagement flow

## Project Highlights

- Full-scale live streaming social platform
- Real-time audience interaction system
- Virtual gifting and engagement features
- Audio/video calling and private chat
- Reels-based short content experience
- Modern and scalable Flutter architecture
- Built for high user engagement and creator interaction

## Project Overview

Havatai was developed as a complete social live streaming platform inspired by modern entertainment and creator-focused applications. It combines live broadcasting, communication, gifting, and social interaction into one polished and interactive mobile experience.

This project demonstrates expertise in building **real-time streaming applications, social engagement systems, live communication features, and scalable Flutter-based mobile platforms**.
